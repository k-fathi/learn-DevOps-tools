## About This Course

This repository contains **31 professional markdown files** covering everything from basic commands to advanced system administration. Each module is structured for both self-paced learning and quick reference.

> ## **Track Your Progress:** visit [DevOps Galaxy](https://devops-galaxy.me) to track your learning journey, find resources, and master DevOps tools.

### Learning Objectives

By the end of this course, you will be able to:
- ✅ Navigate and manage the Linux file system confidently
- ✅ Control users, groups, and file permissions
- ✅ Monitor and manage system processes and services
- ✅ Configure networks and secure SSH access
- ✅ Analyze system logs for troubleshooting
- ✅ Manage software packages on Debian and RedHat systems
- ✅ Write efficient shell scripts and utilize text processing tools

## Quick Start

### Prerequisites
- A Linux machine (physical, virtual, or WSL2)
- Terminal emulator
- Basic computer literacy

### Recommended Learning Path

1. **Beginners:** Start with Module 1 and progress sequentially
2. **Intermediate:** Jump to Modules 4-6 for user/process management
3. **Advanced:** Focus on Modules 7-8 for networking and system admin

### Study Tips
- Type every command yourself (don't copy-paste)
- Experiment in a safe environment (VM recommended)
- Complete the exercises in each module
- Review the [CHEATSHEET.md](./CHEATSHEET.md) regularly
