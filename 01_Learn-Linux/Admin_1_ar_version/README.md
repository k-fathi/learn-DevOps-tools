## عن الكورس ده - About This Course

الـ Repository دي فيها **31 ملف Markdown** بيغطوا كل حاجة من أول الأوامر الأساسية (Basic Commands) لحد إدارة النظام المتقدمة (Advanced System Admin). كل درس متصمم عشان تذاكره لوحدك وكمان يكون مرجع سريع ليك في الشغل.

> ##  تابع تقدمك من خلال [DevOps Galaxy](https://devops-galaxy.me) عشان تسجل كل حاجة اتعلمتها وتلاقي مصادر تساعدك في التعلم.

### هتتعلم إيه؟ - Learning Objectives
<div align="right">

بعد ما تخلص الكورس ده، هتبقى برنس في الحاجات دي:
1. تتحرك وتدير ملفات الـ **Linux File System** بطلاقة.
2. تتحكم في الـ **Users** والـ **Groups** وتظبط الـ **Permissions** بالمللي.
3. تراقب وتدير الـ **Processes** والـ **Services** اللي شغالة على السيرفر.
4. تظبط الشبكات (**Networks**) وتأمن اتصال الـ **SSH**.
5. تحلل الـ **System Logs** وتعرف المشاكل فين (Troubleshooting).
6. تدير الـ **Packages** وتسطب برامج على أنظمة **Debian** و **RedHat**.
7. تكتب **Shell Scripts** تنجزلك شغلك وتستخدم أدوات معالجة النصوص باحترافية.

</div>

## ابدأ بسرعة - Quick Start
<div align="right">

### محتاج إيه عشان تبدأ؟ - Prerequisites
- جهاز عليه لينكس (سواء أساسي، أو Virtual Machine، أو حتى WSL2 على ويندوز).
- برنامج Terminal.
- معرفة بسيطة جداً بالكمبيوتر.

### تذاكر إزاي؟ - Learning Path
1. **لو لسه بتبدأ:** خد الملفات بالترتيب من Module 1.
2. **لو مستواك متوسط:** خش علطول على Modules 4-6 عشان الـ Users والـ Processes.
3. **لو محترف:** ركز على Modules 7-8 للشبكات والـ Admin Tasks التقيلة.

### نصائح للمذاكرة - Study Tips
- اكتب كل أمر بإيدك (بلاش Copy-Paste عشان المعلومة تثبت).
- جرب على سيرفر تجريبي (Test VM) عشان لو بوظت حاجة متبقاش مشكلة.
- طبق التمارين الموجودة في كل درس.
- راجع ملف [CHEATSHEET.md](./CHEATSHEET.md) كل فترة.

</div>
