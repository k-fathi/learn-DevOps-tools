# 28: ملفات النظام والتركيب (FileSystem & Mounting)

## 1. مقدمة
الهارد ديسك في لينكس مش بيظهر كـ C: و D: زي الويندوز. لازم تعمله **Mount** (تركيب) جوه فولدر عشان تقدر تشوفه.
> ![Linux Filesystem Hierarchy](screens/infographic_filesystem.png)

## 2. تقسيم الهارد (Partitioning)
- **MBR:** النظام القديم (أخره 4 بارتيشن، وأخره هارد 2 تيرا).
- **GPT:** النظام الحديث (عدد لا نهائي، ومساحات خرافية).

## 3. أسماء الهاردات
هتلاقيهم في `/dev/`.
- **SATA/USB:** بيبدأ بـ `sd` (زي `/dev/sda` للهارد الأول، `/dev/sdb` للتاني).
    - البارتيشن بياخد رقم: `/dev/sda1`.
- **NVMe:** بيبدأ بـ `nvme` (زي `/dev/nvme0n1`).
- **Virtual:** بيبدأ بـ `vd` (زي `/dev/vda`).

## 4. التركيب (Mounting)
يعني تربط البارتيشن بفولدر عندك (Mount Point).

### تركيب مؤقت (بيروح مع الريستارت)
```bash
# ركب البارتيشن sdb1 جوه الفولدر /mnt/usb
sudo mount /dev/sdb1 /mnt/usb

# شيله (Unmount)
sudo umount /mnt/usb
```

### تركيب دائم (`/etc/fstab`)
الملف ده `/etc/fstab` هو اللي بيقول للجهاز مين يركب فين أول ما يفتح.

> [!DANGER]
> **تحذير هام:** غلطة واحدة في الملف ده ممكن تخلي الجهاز **مش بيفتح**.
> دايماً جرب بأمر `sudo mount -a` قبل ما تعمل ريستارت. لو طلع Error صلحه فوراً.

### عرض المساحة
```bash
df -h      # المساحة الفاضية (Human readable)
lsblk      # شجرة الهاردات والبارتيشنز
du -sh .   # مساحة الفولدر اللي أنا فيه
```

## 5. الزتونة (Summary)
- `mount`/`umount`: ركب واخلع.
- `df -h`: شوف المساحة.
- `/etc/fstab`: التركيب الدائم.

---

## 6. 🏆 مثال من سوق العمل: إضافة هارد جديد للسيرفر
**السيناريو:** ركبت هارد جديد 100 جيجا (`/dev/sdb`) وعايز تشغله وتخليه يركب لوحده كل مرة السيرفر يفتح على `/data`.

```bash
# 1. فرمت البارتيشن بنظام ext4
sudo mkfs.ext4 /dev/sdb1

# 2. اعمل الفولدر اللي هيركب فيه
sudo mkdir /data

# 3. هات الـ UUID (البصمة) بتاعته - أضمن من الاسم
sudo blkid /dev/sdb1
# Output: UUID="a1b2c3d4-..." TYPE="ext4"

# 4. ضيفه في ملف الـ fstab
sudo nano /etc/fstab
# ضيف السطر ده في الآخر:
# UUID=a1b2c3d4-...  /data  ext4  defaults  0  2

# 5. خطوة الموت (Test)
sudo mount -a
# لو مطلعش أي رسالة خطأ، مبروك عليك. لو طلع Error، متقفلش غير لما تصلحه!
```
