# 04: إزاي تطلب مساعدة (Asking for Help)

## 1. مقدمة
لينكس قوي جداً في التوثيق (Documentation). مش دايماً محتاج إنترنت عشان تعرف أمر بيتكتب إزاي أو ملف إعدادات بيشتغل إزاي. الكتالوج كله معاك.

## 2. النجدة السريعة (`--help`)
أغلب الأوامر فيها أوبشن اسمه `--help` أو `-h` بيعرضلك ملخص سريع للأمر بيعمل إيه.

```bash
ls --help
mkdir --help
```

## 3. الكتالوج الكامل (`man`)
الأمر `man` (اختصار Manual) هو المرجع الرسمي لكل حاجة في لينكس.

### الطريقة
```bash
man <command>
```

### أقسام الكتالوج
المانيوال متقسم أبواب (Sections). أهمهم:
1.  **User Commands:** أوامر المستخدم العادية (زي `ls`, `cd`).
5.  **Configuration Files:** ملفات الإعدادات (زي `/etc/passwd`).
8.  **System Admin Commands:** أوامر المديرين (زي `useradd`).

### أمثلة
```bash
# اعرض كتالوج الأمر 'passwd' (تغيير الباسورد)
man passwd

# اعرض شرح ملف '/etc/passwd' (هيكلية الملف) - الباب الخامس
man 5 passwd
```

### إزاي تقرأ وتتحرك جوه الـ man
| الزرار | بيعمل إيه |
| :--- | :--- |
| `Space` | اقلب صفحة |
| `Enter` | انزل سطر |
| `/` | ابحث عن كلمة (اكتب `/word` ودوس Enter) |
| `n` | هات النتيجة اللي بعدها (Next) |
| `N` | هات النتيجة اللي قبلها (Previous) |
| `q` | اخرج (Quit) |

## 4. البحث بالوصف (`-k`)
لو مش عارف اسم الأمر، بس عارف بيعمل إيه، ابحث في الوصف بتاعه بـ `-k` (زي `apropos`).

```bash
man -k "partition"
# هيعرضلك كل الأوامر اللي ليها علاقة بالتقسيم (Partitioning)
```

## 5. صفحات المعلومات (`info` & `pinfo`)
بعض أدوات GNU بتوفر شرح أكتر تفصيلاً بـ `info`.
```bash
info ls
pinfo ls  # (نسخة ملونة وأشيك)
```

## 6. ملفات التوثيق (`/usr/share/doc`)
برامج كتير بتنزل معاها ملفات شرح وأمثلة (READMEs) في المسار ده:
`/usr/share/doc/`

---

## 7. 🏆 مثال من سوق العمل: احتراف الـ `man` pages
**السيناريو:** محتاج تضغط ملفات بـ `tar` بس نسيت إيه الأوبشن اللي بيعمل "Verify" (تأكيد) للملف المضغوط بعد ما يخلص. بدل ما تفتح جوجل، هتجيبها من الـ `man`.

```bash
# 1. افتح المانيوال
man tar

# 2. ابحث عن كلمة "verify" (اكتب كدة جوه المانيوال)
/verify
# دوس 'n' لحد ما تلاقي -W أو --verify

# 3. ابحث عن "gzip" بالمرة
/gzip
# هتلاقي -z

# 4. اخرج
q
```

> **النتيجة:** عرفت إن `tar -W` هو اللي بيعمل Verify، من غير ما تخرج من الترمنال.
