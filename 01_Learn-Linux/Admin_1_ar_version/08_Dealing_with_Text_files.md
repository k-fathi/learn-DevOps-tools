# 08: التعامل مع الملفات النصية (Text Files)

## 1. مقدمة
لينكس مليان أدوات تخليك تقرأ، تعدل، وتكتب ملفات نصية من غير ما تفتح جرافيك.

## 2. قراءة الملفات

### `cat` (القطة اللي بتعرض)
بيعرض محتوى الملف كله مرة واحدة على الشاشة.
```bash
cat /etc/os-release
```

### `more` & `less` (التصفح)
لو الملف كبير، استخدم دول عشان تقرأ صفحة صفحة.
- **`more`**: بدائي شوية (بالنسبة المئوية).
- **`less`**: المتطور (تقدر تطلع وتنزل وتبحث). **أنصح بيه**.

### `head` & `tail` (الراس والديل)
اقرأ أول أو آخر حتة من الملف.
```bash
head -n 5 file.txt   # أول 5 سطور
tail -n 5 file.txt   # آخر 5 سطور
tail -f access.log   # (Follow) راقب الملف وهو بيتكتب لايف (تحفة للـ Logs)
```

## 3. تعديل الملفات (Vim)
**Vim** هو "وحش" المحررات. صعب في الأول بس لو اتعودت عليه هتبقى ساحر.
> **نصيحة:** اكتب `vimtutor` في الترمنال عشان تاخد كورس تفاعلي سريع.

### أ. الأوضاع (Modes)
> ![Vim Modes Quick Reference](screens/infographic_vim_modes.png)

1.  **Command Mode (الأساسي):** للتحرك والأوامر (أول ما تفتح بتلاقي نفسك فيه).
2.  **Insert Mode:** للكتابة (دوس `i`).
3.  **Visual Mode:** للتظليل والنسخ (دوس `v`).
4.  **Extended Mode:** للحفظ والخروج (دوس `:`).

### ب. التحرك (جوه Command Mode)
- `gg`: أول الملف.
- `G`: آخر الملف.
- `/pattern`: ابحث عن كلمة.
- `dd`: قص السطر الحالي (Delete).
- `yy`: انسخ السطر الحالي (Yank).
- `p`: الصق (Paste).
- `u`: تراجع (Undo).

### ج. الحفظ والخروج
- **:w** - احفظ (Save).
- **:q** - اخرج.
- **:wq** - احفظ واخرج.
- **:q!** - اخرج من غير ما تحفظ (لو عكيت الدنيا).

## 4. 🏆 مثال من سوق العمل: سحر الـ Macros في Vim
**السيناريو:** عندك ملف فيه 100 سطر داتا ملخبطة زي `Item: apple`، وعايز تحولها لـ JSON فورمات `{"name": "apple"},`. هتعملها يدوي؟ لا طبعاً.

```bash
# 1. افتح الملف
vim data.txt

# 2. ابدأ تسجيل الماكرو 'q' وسجله في الحرف 'a'
qa

# 3. اعمل التعديل على سطر واحد بس ببطء وتركيز:
# - روح لأول السطر (0)
# - امسح كلمة "Item: " (dw dw)
# - اكتب القوس والبدابة (i {"name": " Esc)
# - روح للآخر واقفل القوس (A "}, Esc)
# - انزل للسطر اللي بعده (j)

# 4. وقف التسجيل
q

# 5. شغل الماكرو 99 مرة
99@a
```

> **النتيجة:** عدلت 100 سطر في ثانية. عشان كدة الـ SysAdmins بيعشقوا Vim.

## 5. الزتونة (Key Takeaways)
- ملف صغير؟ `cat`. ملف كبير؟ `less`.
- بتراقب لوجات؟ `tail -f`.
- لازم تتعلم أساسيات `vim` (على الأقل إزاي تكتب وتخرج `i`, `:wq`).
