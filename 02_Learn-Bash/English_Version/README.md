## About This Course

This repository contains **25 professional markdown files** covering everything from shell fundamentals to advanced scripting, text processing, and task automation. Each module is structured for both self-paced learning and quick reference.

> ## **Track Your Progress:** visit [DevOps Galaxy](https://devops-galaxy.me) to track your learning journey, find resources, and master DevOps tools.

### Learning Objectives

By the end of this course, you will be able to:
- ✅ Understand the relationship between the Kernel, Shell, and Terminal
- ✅ Write and execute Bash scripts with proper environment setup
- ✅ Work with variables, arithmetic operations, and command substitution
- ✅ Implement conditional logic with `if`, `case`, and test operators
- ✅ Master loops (`for`, `while`, `until`) and flow control
- ✅ Use arrays, functions, and file I/O in your scripts
- ✅ Process text with Regular Expressions, `sed`, and `awk`
- ✅ Automate tasks with `cron` and `at` scheduling

## Quick Start

### Prerequisites
- A Linux machine (physical, virtual, or WSL2)
- Terminal emulator
- Basic familiarity with the Linux command line (see [Linux Admin 1](../../01_Learn-Linux/Admin_1/README.md))

### Recommended Learning Path

1. **Beginners:** Start with Module 1 and progress sequentially
2. **Intermediate:** Jump to Modules 3-4 for conditionals and loops
3. **Advanced:** Focus on Modules 5-7 for regex, text processing, and automation

### Study Tips
- Type every command yourself (don't copy-paste)
- Experiment in a safe environment (VM recommended)
- Write small scripts to practice each concept
- Review the [CHEATSHEET.md](./CHEATSHEET.md) regularly
