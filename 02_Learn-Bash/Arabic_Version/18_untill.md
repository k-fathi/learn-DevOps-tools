## 18. حلقة التكرار العكسية (Until Loop)

لوب `until` هي بالظبط **عكس** لوب `while`.
في اللوب اللي فاتت (`while`)، الكود كان بيفضل شغال **"طول ما"** الشرط صح. 
أما في الـ `until`، الكود بيفضل شغال **"لحد ما"** الشرط يبقى صح (بمعنى تاني: بيفضل شغال طول ما الشرط غلط!).

### الصيغة العامة (Basic Syntax)

```bash
until [ condition ]   
do  
    # الأوامر اللي هتتنفذ هنا طالما الشرط غلط (False)
    # أول ما الشرط يتعدل ويبقى صح (True)، اللوب هتقف والإسكربت يكمل
done
```

---

### أمثلة Process

#### مثال 1: عداد بسيط (Simple Counter)
لو عايزين نعمل عداد بيعد من 1 لـ 5:

```bash
count=1

# الإسكربت هيفضل يلف "لحد ما" العداد يبقى أكبر من 5
until [ $count -gt 5 ]
do
    echo "العداد دلوقتي: $count"
    ((count++))
done
```

**النتيجة:**
```
العداد دلوقتي: 1
العداد دلوقتي: 2
العداد دلوقتي: 3
العداد دلوقتي: 4
العداد دلوقتي: 5
```
*(أول ما العداد وصل لـ 6، الشرط `$count -gt 5` بقى صح (True)، فقام الـ `until` موقف اللوب فورا).*

---

#### مثال 2: استخدام الـ Until للانتظار (Waiting for a File)
ده من أقوى استخدامات الـ `until` في الحقيقة، إننا نخلي الإسكربت يعلق نفسه وما يكملش إلا لما ملف معين يتعمل (مثلاً ملف بيتبعت من سيرفر تاني).

```bash
# الإسكربت هيفضل يلف "لحد ما" الملف اللي اسمه ready.txt يبقى موجود (-f)
until [ -f /tmp/ready.txt ]
do
    echo "في انتظار ظهور الملف /tmp/ready.txt ..."
    sleep 2  # استريح ثانيتين وارجع إسأل تاني
done

echo "الملف ظهر! هنكمل شغل دلوقتي."
```

الملف ده هيفضل شغال أوتوماتيك ويقول "في انتظار ظهور الملف.." كل ثانيتين، وأول ما إنت تكريت الملف ده بأمر `touch` في ترمنال تانية، اللوب هتعرف وتقف وتطبع الرسالة وتكمل.
