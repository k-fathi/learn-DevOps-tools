## 8. استخدام جمل `if` الشرطية للتحكم في الكود

في باش، جملة `if` بتسمحلك تاخد قرارات جوه الإسكربت بتاعك، بحيث الكود ينفذ أوامر معينة بس لو الشرط بتاعك طلع صحيح (True).

**الصيغة العامة (Syntax):**
```bash
if [ condition1 ]; then
   # الكود ده هيتنفذ لو الشرط الأول صح
elif [ condition2 ]; then
   # الكود ده هيتنفذ لو الشرط الأول غلط بس الشرط التاني صح
else
   # الكود ده هيتنفذ لو كل الشروط اللي فوق طلعت غلط
fi
```
*(ملاحظة: بننهي الـ `if` بكلمة `fi` اللي هي عكس كلمة if).*

**طريقة تانية للكتابة (بدون سيمي كولون `;`):**
```bash
if [ condition1 ]
then
   # الكود
elif [ condition2 ]
then
   # الكود
else
   # الكود
fi
```
تقدر تختار أي ستايل يعجبك حسب اللي يريح عينك أثناء القراءة.

---

### أمثلة Process على استخدام `if`

**مثال 1: التأكد من إن ملف معين موجود**
```bash
if [ -f "myfile.txt" ]; then
    echo "الملف موجود."
else
    echo "الملف مش موجود."
fi
```

**مثال 2: استخدام `elif` عشان نفحص كذا شرط لرقم**
```bash
read -p "أدخل رقم: " num
if [ "$num" -gt 0 ]; then
    echo "رقم موجب"
elif [ "$num" -lt 0 ]; then
    echo "رقم سالب"
else
    echo "صفر"
fi
```

### استخدام أمر `command` المدمج
كلمة `command` دي أداة مدمجة في باش (Built-in) ببنستخدمها عشان نجبر التيرمينال إنها تنفذ أمر خارجي فعلي من النظام، وتتجاهل أي `alias` أو `function` (Function مؤقتة) إنت عاملها بنفس الاسم.

**الصيغة:**
```bash
command [options] command_name [arguments]
```

**مثال:**
```bash
command ls -l
```
هنا إنت بتقوله نفذ أمر `ls -l` الحقيقي بتاع اللينكس، حتى لو إنت كنت عامل Function (function) جوه الإسكربت بتاعك مسميها `ls` بالغلط.

**أهم الاختيارات (Options) معاه:**
- `-v`: بيعرض Path مكان الأمر فين بالظبط في الجهاز.
- `-V`: بيعرض تفاصيل أكتر عن الأمر ده، هل هو أمر مبني جوه الـ Shell ولا خارجي وملفه فين.
- `-p`: بيستخدم الـ `$PATH` الافتراضي عشان يدور على الأمر ويهمل أي تعديلات إنت عملتها في المسارات.

**مثال تطبيقي: التأكد هل برنامج معين متسطب ولا لأ**
بمساعدة الـ `if` و `command` نقدر نعرف هل في برنامج موجود قبل ما نشغله وريتنا سيستم إيرور ولا لأ:
```bash
if command -v git 1&2> /dev/null; then
    echo "برنامج الغيت (Git) متسطب وجاهز."
else
    echo "البرنامج مش موجود على الجهاز."
fi
# استخدمنا هنا `1&2> /dev/null` عشان نكتم أي أوت بوت بيطلعه أمر الفحص على الشاشة، عشان الفحص يكون صامت (Silent Check).
```

ولو حبينا نشيل كتم الصوت عشان نشوف نتيجة الفحص بعنينا:
```bash
if command -v git; then
    echo "برنامج الغيت متسطب."
else
    echo "البرنامج مش موجود."
fi
# الطباعة هتكون:
   # /usr/bin/git
   # برنامج الغيت متسطب.
```
