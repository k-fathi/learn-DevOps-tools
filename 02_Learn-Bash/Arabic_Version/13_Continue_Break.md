## 13. التحكم في Path اللوب (`continue` و `break`)

وأنت بتلف جوه حلقة (Loop) زي `for` أو `while`، أحياناً بتحتاج تتحكم في اللفة نفسها: إما إنك تنط لفة معينة وتكمل، أو إنك تكسر اللوب وتخرج منها خالص.

### أولاً: أمر `continue` (تجاوز اللفة الحالية)
أمر `continue` بيستخدم عشان "ينط" أو يتجاهل الكود اللي باقي في اللفة دي، ويرجع من الأول تاني عشان يبدأ اللفة اللي بعدها.

**مثال لتخطي رقم معين:**
```bash
for i in {1..5}
do
    if [ $i -eq 3 ]; then 
        continue
    fi
    echo "الرقم هو: $i"
done
```
**النتيجة:** (هتلاقيه طبع كله ونط رقم 3)
```
الرقم هو: 1
الرقم هو: 2
الرقم هو: 4
الرقم هو: 5
```

**مثال تاني: طباعة الأرقام الفردية بس (تخطي الزوجي):**
```bash
for i in {1..6}
do
    if [ $((i % 2)) -eq 0 ]; then
        continue
    fi
    echo "رقم فردي: $i"
done
```
*(هيطبع 1 و 3 و 5 بس).*

---

### ثانياً: أمر `break` (كسر اللوب نهائياً)
بيستخدم عشان تخرج بره الـ Loop خالص ومكملش لف، حتى لو لغة اللوب مخلصتش.

**مثال للخروج عند رقم معين:**
```bash
for i in {1..5}
do
    if [ $i -eq 3 ]; then 
        break
    fi
    echo "الرقم هو: $i"
done
```
**النتيجة:** (طبع 1 و 2، وأول ما وصل لـ 3 كسر اللوب وخرج).
```
الرقم هو: 1
الرقم هو: 2
```

**مثال تقيل: إيقاف السكريبت أول ما تلاقي رقم سالب:**
```bash
numbers="4 7 -2 8 3"
for n in $numbers
do
    if [ $n -lt 0 ]; then
        echo "لقيت رقم سالب، هكسر اللوب دلوقتي! الرقم هو: $n"
        break
    fi
    echo "الرقم سليم: $n"
done
```

### معلومة للمحترفين: التعامل مع الـ Nested Loops (اللوب المتداخلة)

لما يكون عندك لوب جوه لوب (مثلاً بتلف على صفوف وبعدها عواميد)، أوامر `continue` و `break` بتطبق بس على **أصغر وأقرب لوب** مكتوبة جواها (Innermost loop).

طب لو عايز تكسر اللوب اللي بره خالص من جوه؟ عندك طريقتين:

- **1. استخدام الـ Flags (كمتغير وظيفته إشارة):**
```bash
found=false
for i in {1..3}
do
   for j in {1..3}
   do
      if [ $i -eq 2 ] && [ $j -eq 2 ]; then
         found=true
         break
      fi
      echo "i: $i, j: $j"
   done
   # اللوب الكبيرة بتسأل الـ Flag لو حصل الكسر جوه
   if [ "$found" = true ]; then
      break
   fi
done
```

- **2. تحديد كام لوب تتكسر مع أمر Break:**
بباش بتسمحلك تكتب `break n` بحيث الـ `n` دي عدد اللوبات اللي عايز توقفها. يعني لو كتبت `break 2` هيخرج من اتنين لوب، ولو `break 3` هيخرج من تلاتة.
```bash
for i in {1..3}
do
   for j in {1..3}
   do
      if [ $i -eq 2 ] && [ $j -eq 2 ]; then
         break 2  # كده هيكسر اللوب الداخلية وكمان الخارجية مرة واحدة
      fi
      echo "i: $i, j: $j"
   done
done
```
